cd nameورود به اون دایرکتوری مورد نظر
هر فولدری باشی.git init بزنی از اون فولدر یه فایل .gitمیسازه
با touchمیتونی یه فایل  جدید باز کنی توی اون فولدر
بعدش باید git add بزنی که فایل تکست یا هرچی که باز کردی به به مرحله استیج واسه کامیت کردن
برای ساختن یه شاخه دیگه از git branch namebranch استفاده کن و سعی کن بین اسمت فاصله نذاری.
برای اینکه بفهمی رو کدوم شاخه هستی از git branch استفاده کن
برای جابجایی بین شاخه ها از git checkout namebranchاستفاده کن.
برای اضافه کردن تغییرات هر شاخه به شاخه اصلی از git merge namebranchاستفاده میکنیم .
از cat namefileبرای نشان دادن محتوای داخل فایل بهره میبریم.
برای حذف کردن branchها از git branch -d namebranchاستفاده میکنیم.
برای مشاهده تغییرات هر فایل از git diff HEAD استفاده میشود.
برای مشاهده تغییرات فایل های در مرحله استیج از git diff --staged استفاده میکنیم.که متاسفانه مثل بالا مشکل خارج نشدن برامون پیش میاد.
برای خارج کردن فایل از حالت استیج از git reset namefile استفاده میکنیم.
برای برگرداندن یه فایل به حالت اولیه از git checkout -- namefile استفاده میکنیم که حتی بعد از commit شدن هم باز فایل را به حالت قبل برمیگرداند.
